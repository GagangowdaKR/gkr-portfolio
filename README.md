**Portfolio App**
May be Mine :)

A professional (Joke) portfolio application built with React Native and Expo, designed to run seamlessly on Android, iOS, and Web platforms.

**Prerequisites**
Knowledge and Time

**Before you begin, ensure you have the following installed:**
Node.JS v18+ (npm/npx), Expo CLI, Android Studio (Android OS), Xcode (IOS OS), Any browser (Web view) 

**Installation**
Refer Youtube 

**Running the Application**
Start Development Server : npm start
   This will start the Expo development server and display a QR code in your terminal.

**Run on specific platforms**
Android : npm run android
iOS : npm run ios
Web : npm run web
   Alternative: Expo Go App (Try to use latest version else don't try)
