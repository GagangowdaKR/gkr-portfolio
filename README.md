**Portfolio App**

A professional portfolio application built with React Native, Expo and Spring, designed to run seamlessly on Android, iOS, and Web platforms.

**Prerequisites**
Node.JS v18+ (npm/npx), Expo CLI, Android Studio (Android OS), Xcode (IOS OS), Any browser (Web view) 

**Running the Application**
Start Development Server : npm start
   This will start the Expo development server and display a QR code in your terminal.

**Run on specific platforms**
Android : npm run android
iOS : npm run ios
Web : npm run web
   Alternative: Expo Go App (Try to use latest version)
